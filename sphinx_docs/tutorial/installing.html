Installing Cork¶

Before you install¶

Cork, like many things has dependencies. All are in this list below:

  • Docker 1.12+

Yep, that's just one dependency ;)

Install from binary¶

To install cork, download the latest binary for your platfrom from here and place it on your path

Installing from source¶

Prerequisites¶

  • Golang 1.8+
  • A properly configured golang environment

Build from source¶

Pull down the source into a path.

$ go get github.com/virtru/cork
